### Database Creation Workflow
1. User publishes website â†’ Database auto-created
2. OR user manually creates database (for testing)
3. Database name: `pagevoo_website_{user_id}`
4. Core migrations run first (installed_features table)
5. Feature migrations run when feature installed

### Feature Installation Workflow
1. User clicks Insert > Feature > [Feature Name]
2. Check if database exists:
   - If NO â†’ Prompt to publish or create database
   - If YES â†’ Continue
3. Show configuration modal
4. Save configuration to database
5. Run feature-specific migrations
6. Create auto-generated pages (if applicable)
7. Add feature to installed_features table
8. Success message + redirect to feature admin
